# 静态图片资源说明

目录：`prototype/static/images/`

用于存放原型阶段的图片资源与占位图。

## 1. 建议规格
- 商品图：400x400
- 轮播图：1920x400
- 分类图标：64x64
- Logo：200x80

## 2. 建议格式
- 商品图：JPG/WebP
- 图标与 Logo：PNG/SVG

## 3. 命名建议
- `product-001.jpg`
- `banner-001.jpg`
- `category-digital.png`
- `logo.png`

## 4. 说明
当前运行中的 Flask 页面主要使用数据库中的图片 URL，
本目录保留给原型与前端视觉设计阶段使用。
